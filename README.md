# Event Booking System
## Описание
Данный проект представляет собой веб-приложение для управления мероприятиями и бронирования билетов.  
Пользователи могут просматривать список мероприятий, получать подробную информацию и осуществлять бронирование.

Приложение разработано на базе Spring Boot с использованием архитектуры MVC.

## Функциональные возможности
- Регистрация и аутентификация пользователей
- Просмотр и фильтрация мероприятий
- Просмотр деталей мероприятий
- Бронирование и отмена бронирований
- Добавление и удаление мероприятий
- Управление справочными данными (жанры, залы)
